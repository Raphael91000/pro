# Instructions pour Claude

## Workflow

- Code **directement sur les fichiers locaux** — pas de worktrees, pas de branches temporaires, pas d'environnements isolés
- Modifie les fichiers en place, sans détour

- Tu interviens **uniquement sur le frontend**
